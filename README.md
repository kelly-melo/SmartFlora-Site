# SmartFlora-Site

Este repositório contém o código-fonte do site de documentação do projeto SmartFlora, um sistema de irrigação automatizada de plantas baseado em Arduino.

## Como Visualizar o Site

O site é composto por arquivos HTML, CSS e JavaScript estáticos. Para visualizá-lo, você não precisa de um servidor web complexo. Siga os passos abaixo:

1.  **Clone ou baixe este repositório** para o seu computador.
2.  **Navegue até a pasta** onde você salvou os arquivos.
3.  **Abra o arquivo `index.html`** diretamente no seu navegador de preferência (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).

Ao abrir o arquivo `index.html`, o site será carregado e você poderá navegar por todas as seções, incluindo a introdução ao projeto, o guia de montagem, o código-fonte e as informações da equipe.

## Como Usar o Código Arduino

O código-fonte para o microcontrolador Arduino está disponível na página "Código" do site. Para utilizá-lo:

1.  **Acesse a página `codigo.html`** navegando pelo menu do site ou abrindo o arquivo diretamente.
2.  **Copie o código** da seção "Código Arduino (C/C++)".
3.  **Abra a sua Arduino IDE**.
4.  **Cole o código** em um novo sketch (ou em um sketch existente).
5.  **Conecte sua placa Arduino** ao computador, selecione a placa e a porta corretas no menu "Ferramentas" da IDE.
6.  **Carregue (upload)** o código para a placa.
